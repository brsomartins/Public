Assignment 7

  1. For this assignment, you will be creating an HTML page with JavaScript functions and classes which mimic the functionality of your assignments 2 and 3. That is, you will be creating Exam, Coursework, and Student classes which can be used to compute the final course letter grade and percentage score for a student in a hypothetical class. You must write your page "by hand", meaning that you are not allowed to use an HTML editor or application which writes HTML for you, and all code should be contained within a single file, which is the only file you will submit.


  2. Create a file named FirstLast.html (where "First" is replaced with your first name and "Last" with your last name). Create a title for your page (using the "title" tag) and a main heading (using the "h1" tag) with the text "First Last's Assignment 7" (again substituting your name).


  3. Other than the heading, your page should only display a form which requests the student's data. The form should contain 8 text input fields and one "Submit" button. The input fields should request:
    1. first name
    2. last name
    3. a comma separated list of homework scores (e.g. "90,80,98")
    4. a comma separated list of activity scores
    5. a midterm exam percentage score (e.g. "91")
    6. a midterm exam letter grade (e.g. "A")
    7. a final exam percentage score
    8. a final exam letter grade
    There should be text next to each input field specifying which data to enter into it. Note: For easier testing, you can set your input text fields to have default test values, and obviously these will be changed by the grader when grading.


  4. Set your form so that when the submit button is pressed, a JavaScript function named "calculateGrade" is called. Hint: Use the "onsubmit" attribute of the form as follows: onsubmit="return calculateGrade()". Also, make sure that your "calculateGrade" function always returns false. This will prevent the browser from "submitting" the form and then reloading the page after the submit button is pressed. Make your "calculateGrade" function do something basic like call alert and display data from the form (the name entered, etc.) to ensure that you've got everything working up to this point. Don't continue until you do.


  5. Create JavaScript classes Exam, Coursework, and Student which have the same attributes and methods as those classes did in assignment 2. They should be structured the same (e.g. the Student class should contain 2 Exam and 2 Coursework objects). Further, all attributes of each class should be "private" and the methods should all be "priveleged" (see http://javascript.crockford.com/private.html for details).


  6. Modify "calculateGrade" so that it gathers all of the necessary data from the form and uses it to create a Student object and set all necessary score and grade values. Have it call alert with the string returned by the "getGrade" method of the Student class. Make sure this is working before continuing (including the correct arithmetic for computing final letter grades along with any relevant curves, etc.).


  7. Make one more set of modifications to "calculateGrade" so that instead of displaying the student's grade in an alert popup, it prints that information to the HTML page. The heading should remain visible on the page, but the entire form should now be hidden. Thus, when the page is loaded and the student's information is added into the form and then the submit button is pushed, the form should disappear and the correct grade for the student should be displayed instead (in the format specified for assignment 2, i.e. "Final course grade for First Last is: X (yy%)"). Don't worry about rounding the percentage score to two decimal places or about the spacing on your page, meaning that it doesn't matter where the output information about the student is located, as long as the form is not visible and the heading is.


  8. Here is the rubrick the grader will use, so make sure your page meets each of these requirements:
    1. (-20 points) If anything other than a single, uncompressed HTML file was submitted.
    2. (5 points) Does the HTML file have the correct name?
    3. (5 points) Does the page have the correct title?
    4. (5 points) Does the page have the correct heading?
    5. (15 points) Is the form correctly displayed with all necessary fields (and descriptions of the data to enter next to each) and one button?
    6. (10 points) Does clicking on the button cause the form to disappear?
    7. (10 points) Does clicking on the button cause text to appear which gives the student's name, grade, and score?
    8. (10 points) Is the score computed correctly for all test cases?
    9. (10 points) Is the grade computed correctly for all test cases?
    10. (20 points) Does the JavaScript code correctly define and make use of Exam, Coursework, and Student classes?
    11. (10 point) Are the attributes of classes "private" and the methods "privileged"?


  9. Submit a single HTML file in the usual manner. It should not be zipped or in a compressed file. Test your file before you submit it by copying it to a new folder and opening it in your Chrome web browser from that folder. Ensure that it works as expected.
